# Assets

All client materials — visual, written, and documents.

**What goes here:**
- Logo files (SVG preferred, PNG/JPG as fallback)
- Brand photos / team photos / product images
- Background videos
- Icons or illustrations
- Brand guidelines (colors, fonts, tone of voice)
- Content documents (text for sections, bios, service descriptions)
- PDFs (brochures, flyers, presentations — the agent reads these)
- Pricing info / service packages
- Legal pages (imprint, privacy policy, terms)
- SEO keywords or marketing strategy docs

**Tips:**
- SVG for logos whenever possible
- High-res images — the agent will convert to WebP during build
- Name files clearly (e.g. `logo-dark.svg`, `team-michael.jpg`, `hero-background.mp4`)
- The agent extracts real content from PDFs — no lorem ipsum needed
