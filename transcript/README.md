# Transcript

Drop the onboarding call transcript here. This is the primary brief for the agent.

**What goes here:**
- Full transcript from the client onboarding call
- Any follow-up call transcripts
- Voice memo transcriptions
- Chat/message exports with the client

**Tips:**
- The more detail, the better — the agent picks up on tone, preferences, rejected ideas
- If there are multiple calls, name them in order (e.g. `01_onboarding.txt`, `02_followup.txt`)
